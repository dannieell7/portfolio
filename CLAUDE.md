# Portfolio Website

## Project Overview
A personal portfolio website built with plain HTML, CSS, and JavaScript (no frameworks). Static site showcasing personal projects, skills, and experience.

## Tech Stack
- HTML, CSS, JavaScript (vanilla)
- No build tools or frameworks

## Color Palettes

### Dark Mode — "Blue Eclipse"
- `#272757` — deep indigo
- `#8686AC` — muted lavender
- `#505081` — mid purple
- `#0F0E47` — near-black navy

### Light Mode — "Stormy Morning"
- `#6A89A7` — steel blue
- `#BDDDFC` — pale sky blue
- `#88BDF2` — medium sky blue
- `#384959` — dark slate

## Sections
1. **Hero / Main** — intro, nav links top-right
2. **Projects** — horizontal scroll row of project cards
3. **Skills** — icon row, icons light up on hover
4. **Experience** — cards with experience details
5. **About Me** — personal description
6. **Contact** — header at bottom, contact info bottom-right

## Features
- Light/Dark mode toggle (icon top-right corner)
- Scroll progress bar (visible at top when scrolling)
- Smooth scroll to section when nav links clicked
- Projects: horizontal scrollable row
- Skills: icon grid with hover glow/highlight effect
- Experience: card layout
